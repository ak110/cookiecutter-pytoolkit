cookiecutter-pytoolkit
======================

コンペなど用のテンプレート

Requirements
------------
Install `cookiecutter` command line: `pip install cookiecutter`    

Usage
-----
Generate a new Cookiecutter template layout: `cookiecutter gh:ak110/cookiecutter-pytoolkit`
